# jun-nvim

This repository holds my nvim configuration, originally modified from kickstart. I manage plugins with Lazy

Main plugins:
- Telescope (search)
- Blink (autocomplete)
- nvim-dap (debugging)
- Houdini (keymaps, esc->jk etc)
- lazydev, lspconfig
- indent-blankline, nvimlint, conform
- autopairs, gitsigns, todo-comments
- Neo-tree (occasinally use the file tree)
